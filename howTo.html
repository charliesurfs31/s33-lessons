Styling with CSS

CSS (Cascading Style Sheets) is used to style the layout and appearance of HTML elements on a webpage.

  • External CSS: Create a separate CSS file and link it to your HTML file using the <link> tag.
  • Internal CSS: Use the <style> tag within the <head> section of your HTML file to define CSS rules.
  • Inline CSS: Apply CSS styles directly to HTML elements using the style attribute.

Color

Color is an important aspect of web design. CSS provides various ways to specify colors:

  • Named Colors: Use predefined color names like red, blue, green, etc.
  • Hexadecimal: Specify colors using hexadecimal values like #ff0000 for red.
  • RGB: Define colors using RGB values like rgb(255, 0, 0) for red.
  • RGBA: RGBA allows you to specify a color with an additional alpha channel for transparency, like rgba(255, 0, 0, 0.5).

Size

Controlling the size of elements is crucial in web design to achieve the desired layout. CSS offers various units to specify size:

  • Pixels (px): Fixed-size units that render at a consistent size regardless of the screen resolution.
  • Percentage (%): Relative units that define sizes in relation to the parent element.
  • Viewport Width/Height (vw/vh): Units that represent a percentage of the viewport width or height.
  • Em: Relative unit based on the font-size of the parent element.
  • Rem: Similar to Em, but based on the font-size of the root element.

Adding Animations

CSS allows you to add animations to your webpage to make it more dynamic and interactive.

  • @keyframes: Define animations using the @keyframes rule.
  • animation property: Apply animations to elements using the animation property.
